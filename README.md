# 2D Management Sim - Web

Tato verze bezi primo v prohlizeci a je vhodna pro GitHub Pages.

## Lokalne spusteni

Staci otevrit `index.html` v prohlizeci.

## Publikace na GitHub Pages

1. Vytvor novy GitHub repozitar (napr. `management-game-web`).
2. Nahraj obsah teto slozky (`index.html`, `style.css`, `game.js`, `README.md`) do rootu repozitare.
3. Na GitHubu otevri:
   - `Settings` -> `Pages`
4. V `Build and deployment` nastav:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` a slozka `/ (root)`
5. Uloz nastaveni a pockej cca 1-2 minuty.
6. GitHub vygeneruje verejny odkaz ve tvaru:
   - `https://TVE_JMENO.github.io/management-game-web/`

Hra pak bude zdarma verejne pristupna komukoliv s odkazem.
